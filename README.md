# 공부 유형 연구소

12개의 질문으로 알아보는 나만의 공부 유형 테스트

## 실행 영상

https://github.com/user-attachments/assets/0ada9043-dae2-4d6d-bf92-bd942c87218f

## 기술 스택

- React 18
- Vite
- CSS

## 공부 유형

계획성 · 학습방식 · 환경 3가지 기준으로 총 8가지 유형 제공

| 유형 | 이름 |
|------|------|
| PCI | 계획형 독학러 |
| PCG | 계획형 토론러 |
| PEI | 계획형 문제풀이러 |
| PEG | 협동형 공부러 |
| SCI | 탐구형 독학러 |
| SCG | 토론형 공부러 |
| SEI | 벼락치기형 공부러 |
| SEG | 자유형 스터디러 |

## 주요 기능

- 12문항 테스트
- 유형별 추천 공부법 및 주의할 점 제공
- 테스트 결과 기록 저장 (localStorage)
